# Analisis acciones de Netflix
Este proyecto tiene como objetivo analizar el comportamiento histórico de las acciones de Netflix (NFLX) utilizando técnicas de análisis exploratorio de datos (EDA), visualización, análisis temporal y modelos predictivos básicos.  
El dataset contiene información histórica bursátil de Netflix, incluyendo:  
- Fecha Precio de apertura (Open)
- Precio máximo (High)
- Precio mínimo (Low)
- Precio de cierre (Close)
- Precio ajustado (Adj Close)
- Volumen de operaciones (Volume)
